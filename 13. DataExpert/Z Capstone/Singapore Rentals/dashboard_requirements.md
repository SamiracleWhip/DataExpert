# Requirements

- As a User I am looking for a good deal that has come up in a certain zone or building.
- As a User I want to be able to search trends across Singapore and be able to visualize general trends and be able to drill down to specific ones.
- As a User I want to catch any changes over time to rental prices and so time series visualizations are important to me.
- As a User it is important that I be able to filter by all possible dimensions so that I can find what I want to precision.
- As a User I like a sleak and elegant modern design for my dashboards, including the option to switch from light to dark mode. 
- As a user I want to be able to see graphs as well as tables, although graphs are far more important to me.
